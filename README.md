# Eatly — Enjoy foods all over the World

Веб-приложение для просмотра рецептов и чтения кулинарного блога. Реализовано на React + TypeScript с использованием Redux Toolkit и RTK Query.

## Демо

[eatly-tchigi.vercel.app](https://eatly-tchigi.vercel.app/)

## Стек технологий

**Фреймворк:** React 18, TypeScript 5, Vite 5\
**Маршрутизация:** React Router v6\
**Стейт-менеджмент:** Redux Toolkit, RTK Query\
**Стилизация:** CSS Modules, Google Fonts\
**Тестирование:** Vitest, Testing Library\
**Документация:** Storybook 7

## Функциональность

- Лента блога с пагинацией
- Страница отдельного поста с комментариями
- Добавление комментариев к постам
- Топ рецептов с рейтингом и временем приготовления
- Раздел покупок и статусов заказов
- Отзывы покупателей (слайдер)
- FAQ с раскрывающимися пунктами
- Страница 404
